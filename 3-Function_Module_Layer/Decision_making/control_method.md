# 控制方式说明

通过 `USE_SBUS_RECEIVER` 宏选择控制来源：
- `0`（默认）：DJI 遥控器（DR16），支持遥控器模式和键鼠模式切换
- `1`：SBUS 接收机
- `2`：图传链路（VT03/VT13）

---

## 一、遥控器控制（RC_ctrl_set）

### DJI 遥控器（USE_SBUS_RECEIVER == 0）

| 输入 | 功能 |
|------|------|
| 左摇杆 上/下 | 底盘前后（vx） |
| 左摇杆 左/右 | 底盘横移（vy） |
| 右摇杆 左/右 | 云台 Yaw |
| 右摇杆 上/下 | 云台 Pitch |
| 右拨杆 下 | 底盘跟随云台模式 |
| 右拨杆 中 | 底盘自由模式 |
| 右拨杆 上 | 底盘小陀螺模式 |
| 左拨杆 下 | 遥控器控制模式 |
| 左拨杆 上 | 键鼠控制模式 |
| 拨轮 > 150 | 开启摩擦轮 |
| 拨轮 > 300 | 单发 |
| 拨轮 < -300 | 急停 |

### SBUS 接收机（USE_SBUS_RECEIVER == 1）

| 通道 | 功能 |
|------|------|
| Ch2（左摇杆上下） | 底盘前后 |
| Ch4（左摇杆左右） | 底盘横移 |
| Ch1（右摇杆左右） | 云台 Yaw |
| Ch3（右摇杆上下） | 云台 Pitch |
| Ch8（SB）下/中/上 | 跟随 / 自由 / 小陀螺 |
| Ch9（SC）> 300 | 开启摩擦轮 |
| Ch7（SA）< -300 | 反转拨弹 |
| Ch12（SH）拨下 + SA中 | 单发 |
| Ch12（SH）拨下 + SA上 | 连发（8发/s） |
| Ch5（SF）下 | 急停 |

### 图传链路（USE_SBUS_RECEIVER == 2）

| 输入 | 功能 |
|------|------|
| 左摇杆 上/下 | 底盘横移 |
| 左摇杆 左/右 | 底盘前后 |
| 右摇杆 左/右 | 云台 Yaw |
| 右摇杆 上/下 | 云台 Pitch |
| mode_switch C/N/S | 跟随 / 自由 / 小陀螺 |
| 扳机键 | 开启摩擦轮 + 单发 |
| pause 键 | 急停 |

---

## 二、键鼠控制（Keyboard_ctrl_set）

适用于 DJI 遥控器（左拨杆打上）和图传链路。

| 输入 | 功能 |
|------|------|
| W / S | 底盘前进 / 后退（±3000） |
| A / D | 底盘左移 / 右移（±3000） |
| 鼠标移动 | 云台 Yaw（左右）/ Pitch（上下） |
| 鼠标左键 | 发射（单发或连发，取决于 E 的状态） |
| F | 摩擦轮开关（toggle） |
| E | 切换射击模式：单发 ↔ 连发（toggle） |
| C | 切换底盘模式：跟随 → 自由 → 小陀螺（循环） |

> 连发模式下发射频率为 8 发/s。
