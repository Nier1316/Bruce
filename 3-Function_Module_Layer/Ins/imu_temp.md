# IMU 恒温控制（Temperature Control）模块使用说明文档
## 1. 模块简介

本模块实现了一个 **IMU 恒温控制机制**，用于通过 PWM 控制加热电阻，使 IMU（如 BMI088）工作在稳定温度（默认 40℃），从而减少温漂与零漂，提高姿态解算精度。

## 2. 模块功能介绍

主要功能包括：
- 初始化温控 PID 控制器参数
- 输出 PWM 信号驱动加热电阻
- 根据 IMU 实时温度进行闭环控制
- 支持死区控制，防止抖动
- 支持输出限幅与积分限幅
- 提供统一的温控执行接口

---

## 3. 使用流程

### 3.1 初始化温控模块

在系统初始化阶段，调用初始化函数：

```c
void Imu_Temp_Init(void);
```
### 3.2 周期性执行温控控制
必须在固定周期任务中调用：(建议调用频率：50Hz ~ 100Hz)
```c
void Imu_Temp_Control(float current_temp);
```
