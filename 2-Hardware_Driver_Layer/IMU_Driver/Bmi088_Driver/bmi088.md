# BMI088 驱动模块

## 1. 模块简介

本模块为 **BMI088 六轴 IMU 驱动程序（加速度计 + 陀螺仪）**，基于 SPI 接口直接操作硬件寄存器，完成初始化、数据读取与姿态解算（EKF）。

---

## 2. 对外接口

```c
const Ins_driver_interface_t* BMI088_Get_Driver(void);
float BMI088_Get_Temp_Raw(void);
```
驱动接口结构：
```c
typedef struct {
    bool (*init)(void);
    void (*start_read)(void);
    bool (*wait_data)(void);
    void (*process_data)(Ins_data_t *out_data, float dt_s);
} Ins_
```

## 3. 使用流程
### 3.1 获取驱动对象
```c
const Ins_driver_interface_t *imu = BMI088_Get_Driver();
```
### 3.2 初始化
```c
imu->init();
```
### 3.3 读取与处理数据
```c
imu->start_read();     // 读加速度计
imu->wait_data();      // 读陀螺仪

Ins_data_t data;
imu->process_data(&data, dt);
```

## 4. 使用示例
```c
const Ins_driver_interface_t *imu = BMI088_Get_Driver();
imu->init();

while (1)
{
    imu->start_read();
    imu->wait_data();

    Ins_data_t data;
    imu->process_data(&data, 0.001f);

    //...
}
```
